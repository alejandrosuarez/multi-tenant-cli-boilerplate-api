Key Endpoints

POST /api/auth/send-otp

Send a one-time password to user's email

POST /api/auth/verify-otp

Verify OTP and receive authentication token

GET /api/entities

List entities with optional filtering

POST /api/entities

Create a new entity

POST /api/entities/:id/images

Upload images to an entity

GET /api/entities/search

Advanced entity search with filtering

POST /api/notifications/subscribe-device

Subscribe device for push notifications

POST /api/notifications/send

Send push notification to a user

Getting Started

  1. Visit the API Documentation to explore all available endpoints
  2. Use the "Authorize" button to authenticate with your token
  3. Try out endpoints directly in the browser with the interactive playground
  4. View request/response examples and schema definitions

Authentication Flow Example

# 1. Send OTP to your email POST /api/auth/send-otp { "email": "your.email@example.com" } # 2. Verify OTP and get token POST /api/auth/verify-otp { "email": "your.email@example.com", "otp": "123456" } # 3. Use the token in the "Authorize" dialog # Click the "Authorize" button and enter: # Bearer your-token-here
